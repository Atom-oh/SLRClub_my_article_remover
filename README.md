# SLRClub_my_article_remover

Remove my history

## Requirement

### 가상환경 설정
virtualenv가 없다면 먼저 설치
```sh
python3 -m pip install virtualenv
```
```sh
virtualenv env
```


### 가상환경에서 패키지 설치


```sh
source env/bin/activate
pip install -r requirements.txt
```

## 실행
```sh
python slrclub_remover.py
```
